§ 1. Запуск
После успешной установки программы LanCalculator в вашу систему, вы сможете её запустить:

1) кликнув значок на десктопе

2) выбрав соответствующий пункт в меню Пуск (Start);

3) либо из быстрого запуска (Quick Launch);

§ 2.1. Главное меню

Главное меню состоит из двух пунктов «Действия» и «Помощь»:

Меню «Действия»

Назначение его пунктов следующее:

  • Рассчитать (Enter) — позволяет рассчитать параметры подсети.
  • Очистить — позволяет очистить произведенные вычисления.
  • Мои параметры — расчет всех значений для ваших параметров.
  • Зарезервированные адреса — показывает диалог с перечнем зарезервированных диапазонов адресов.
  • Маски — в подменю приведены примеры масок подсетей.
    • Маска подсети класса A — устанавливает маску для подсети класса А.
    • Маска подсети класса B — устанавливает маску для подсети класса B.
    • Маска подсети класса C — устанавливает маску для подсети класса C.
    • Информация о масках — показывает диалог с информацией о возможных масках.
  • Выбрать язык — выбирает язык интерфейса программы из установленных.
  • Выход — закрывает приложение LanCalculator.


Меню «Помощь»

Вы можете с помощью данного меню получить дополнительную информацию о программе LanCalculator.

  • Руководство пользователя — руководство пользователя о программе LanCalculator.
  • FAQ — часто задаваемые вопросы.
  • Что нового? — изменения в программе.
  • Открыть Web страницу LanCalculator — Открывает в браузере страницу www.LanTricks.com
  • Написать письмо автору — Вы можете написать письмо автору со своими пожеланиями.
  • О программе... — Открывает диалог со сведениями о версии программы и благодарностями.


§ 2.2. Работа с программой

Чтобы рассчитать диапазон адресов в сети, необходимо ввести адрес и маску, маска может быть введена как префикс. По этим данным будут рассчитаны маска, инверсия маски, префикс, а также минимальный и максимальный адрес в сети, адрес сети, широковещательный адрес. Маска подсети указывается в десятичном формате с разделяющими точками (255.255.255.0) либо в "CIDR notation" RFC 1517 (/25) либо в виде инверсной маски подсети, которая используется в списках доступа (ACL) сетевого оборудования Cisco.
