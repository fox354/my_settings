Гарячі системні клавіші (за замовчуванням):
Double Ctrl => Показати головне вікно програми (з перекладом, якщо є виділений текст)
Ctrl + Q => Перекласти виділений текст і показати у спливаючому вікні
Ctrl + W => Показати вікно словника
Ctrl + E => Прослухати виділений текст

Сполучення клавіш у головному вікні:
Ctrl + Enter => Перекласти текст
Ctrl + N => Очистити поточний переклад
Ctrl + H => Показати історію перекладів
Ctrl+K => Показати віртуальну клавіатуру
Ctrl+I => Змінити напрямок перекладу
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => Перемкнути на наступний/попередній/1..9й сервіс
Shift+Esc => Скидання мов на пару за замовчуванням
Ctrl + Space => Показати меню зі списком варіантів слова, що вводиться (автозаповнення)
F1 => Показати Довідку

Програма має два режими перекладу тексту виділенням мишею (як його увімкнути див. нижче):
  1) Показати іконку (виділяєте текст в будь-якій програмі, з'являється іконка програми біля курсору. Клікаєте на іконку, з'являється спливаюче вікно з перекладом виділеного тексту);
  2) Показати переклад (виділяєте текст в будь-якій програмі, з'являється спливаюче вікно з перекладом виділеного тексту).

Дії:
  - Одиночне натистення на іконці програми в треї включає (відключає) режим перекладу виділенням мишею (див. вище);
  - Подвійне натистення на іконці програми в треї показує головне вікно програми;
  - Подвійне натистення на заголовку спливаючого вікна відкриває головне вікно програми з поточним перекладом;
  - Натистення у спливаючому вікні по назві сервісу змінює поточний сервіс та заново перекладає текст.
